--- P R O J E C T S ---

The Lady Barber

Designed, built, & deployed the website for “The Lady Barber”, a local barbershop.

  • Built with HTML, CSS, and Javascript; hosted on Vercel
  • Worked closely with business owner on features and design
  • Fully responsive design
  • Sticky navbar allows user to traverse page quickly
  • Implemented gallery viewer widget that allows users to quickly flip through photos of the business

Dragon's Brew Cafe

Designed & built a website for a fictional coffee shop.

  • Built with MongoDB, ExpressJS, ReactJS, NodeJS and ChakraUI; hosted on Render
  • Fully responsive design
  • Users can add, remove, or "purchase" items
  • Implemented Get, Post and Delete RESTful API's
  • Implemented notifications with the useToast hook from Chakra UI

Thesis Document Generator

Designed & built an app to generate correctly formatted documents for the UA Graduate School.

  • Built with HTML, CSS, ReactJS, and Material UI; hosted on Firebase
  • Worked on a 4-person team to build the app
  • Form stepper for ease of use
  • Document generation through the use of docx
